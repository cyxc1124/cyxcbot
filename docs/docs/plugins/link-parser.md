# 链接解析

自动识别群聊/好友中的 B 站链接并回复解析结果。

## 触发方式

无需命令，发送含以下内容的普通消息即可触发：

- BV 号视频链接
- 直播间链接
- `b23.tv` 短链
- B 站 QQ 小程序分享

## 回复内容

根据链接类型返回：

- **视频**：封面、标题、UP 主、发布时间、链接
- **直播**：封面、主播名、直播状态、链接

## 配置

在 **Web Admin → 群组** 或 **好友** 页面中，按群/好友单独开关链接解析。

支持配置解析策略（如仅解析视频、仅解析直播等）。

## 注意事项

- 插件以较低优先级监听消息，不会阻断其他功能
- 未开启解析的群/好友中发送链接不会触发
