# 状态查询

允许特定 QQ 号查询机器人运行状态。

:::info 2.0 配置方式
允许查询的 QQ 号在 **Web Admin → 群组/好友** 策略或超级用户设置中配置，不再使用 `STATUS_CHECK_ALLOWED_QQ` 环境变量。
:::

## 命令

| 命令 | 别名 |
|------|------|
| `/status` | `/状态`、`/运行状态` |

## 显示信息

- 机器人运行状态
- 运行时间
- 系统信息
- 内存使用情况
- Python / NoneBot 版本
- 连接状态

具体显示项可在 Web Admin 设置中配置。

## 权限控制

- 只响应有权限的用户，无权限用户发送命令时机器人**完全不会回复**
- 超级用户（`SUPERUSERS`）默认可查询
- 可为特定群/好友单独配置允许名单
- 所有查询与无权限尝试均记录日志

## 依赖

使用 `psutil` 获取系统信息，已包含在 `requirements.txt` 中。
