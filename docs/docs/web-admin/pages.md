# 页面说明

## 仪表盘 `/`

展示机器人运行概览：连接状态、监控任务数量、资源使用等。

## 直播监控 `/live`

- 查看各直播间监控状态（开播/下播、WebSocket 连接等）
- 添加/编辑/删除房间号与目标群/好友映射
- 配置 @全体 等推送选项

## 动态监控 `/dynamic`

- 查看各 UP 主监控状态与最近检查时间
- 添加/编辑/删除 UID 与目标群/好友映射
- 查看监控模式（分散/批量）

## 群组 `/groups`

- 管理机器人已加入的群列表
- 配置群消息守卫（是否响应群消息）
- 配置链接解析策略
- 配置状态查询允许名单

## 好友 `/private`

- 管理好友列表
- 配置私聊守卫
- 配置链接解析与状态查询策略

## 消息模板 `/templates`

自定义各类推送消息的文案模板，支持变量占位符。包括：

- 开播 / 下播通知
- 动态更新通知
- 视频投稿通知

## 设置 `/settings`

| 子页面 | 说明 |
|--------|------|
| 监控参数 | 轮询间隔、WebSocket 开关、截图开关等 |
| B 站账号 | 扫码登录或 Cookie 管理 |
| 机器人 | OneBot 连接与超级用户等 |

## 日志 `/logs`

实时查看机器人运行日志，支持按级别过滤。通过 WebSocket 推送，无需刷新页面。

## 关于 `/about`

显示版本号、构建信息与依赖版本。

## 首次设置 `/setup`

仅在无管理员账户时可用，用于创建初始管理员。

## 登录 `/login`

管理员登录入口。
