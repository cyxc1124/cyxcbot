# Web Admin 初始化

首次启动机器草后，需要通过 Web Admin 完成基础配置。

## 1. 创建管理员账户

访问 `http://<主机>:8081/setup`，设置管理员用户名与密码。此步骤仅在数据库中尚无管理员时可用。

## 2. 登录

完成初始化后，通过 `/login` 登录管理面板。

## 3. 配置 OneBot 连接

在 **设置 → 机器人** 中配置 OneBot 协议端连接信息。确保协议端已配置反向 WebSocket 连接到机器人的 **8080** 端口。

## 4. 配置 B 站账号

在 **设置 → B 站账号** 中登录或填写 Cookie。配置 B 站账号可提高 API 请求成功率，对动态截图、直播监控等功能尤为重要。

支持扫码登录，Cookie 加密存储在数据库中。

## 5. 添加监控映射

根据需求在对应页面添加监控：

- **直播监控**（`/live`）：房间号 → 目标群/好友
- **动态监控**（`/dynamic`）：UP 主 UID → 目标群/好友
- **投稿监控**：在动态监控页面中配置

## 6. 配置权限策略

在 **群组**（`/groups`）和 **好友**（`/private`）页面中：

- 开关群/好友消息响应（守卫）
- 配置链接解析策略
- 配置状态查询允许名单

## 7. 自定义消息模板

在 **消息模板**（`/templates`）页面自定义开播、下播、动态等推送文案。
